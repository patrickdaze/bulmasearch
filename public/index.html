Bulma Search provides search for the Bulma documentation.

How it works: Search on this page or install the Chrome extension to add search directly on the Bulma website.

Chrome Extension

Compatibility: The Chrome extension should be compatible with most Chromium browsers including Avast Secure Browser, Blisk, Brave, Comodo Dragon, Google Chrome, Microsoft Edge (recent versions), Yande, and more.
